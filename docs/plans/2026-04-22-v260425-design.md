# v260425 升级设计文档

## 版本信息
- **版本号**: v260425（日期格式）
- **发布日期**: 2026-04-22
- **升级方向**: 从"企微API拉取模式"升级为"数据库直连模式"

## 架构变更

### 旧架构
```
企微智能表格API ← 分页/重试/缓存 → Flask → 前端
```

### 新架构
```
企微智能表格 ──自动同步──→ MySQL/PGSQL (企微帮我们同步)
                              │
                              ▼
                     Flask + SQLAlchemy
                     ├── 前台查询页面（不变）
                     ├── 管理后台（新增，TailwindCSS）
                     └── SQLite（管理数据：用户/数据源/设置）
```

## 核心变化

| 项目 | 旧方式 | 新方式 |
|------|--------|--------|
| 数据来源 | 企微API实时拉取 | MySQL/PGSQL直接查询 |
| 数据同步 | 自己写分页+MD5比对 | 企微自动同步到数据库 |
| 配置管理 | .env文件 | SQLite + 管理后台Web界面 |
| 多数据源 | 不支持 | 支持多个数据库连接 |
| 访问控制 | token参数 | 管理员账号密码登录 |

## 数据库设计

### SQLite（管理库）
```sql
-- 用户表
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    display_name TEXT,
    is_admin BOOLEAN DEFAULT 0,
    last_login TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 数据源表
CREATE TABLE datasources (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    db_type TEXT NOT NULL,       -- mysql / postgresql
    host TEXT NOT NULL,
    port INTEGER NOT NULL,
    database_name TEXT NOT NULL,
    username TEXT NOT NULL,
    password TEXT NOT NULL,       -- AES加密存储
    table_name TEXT NOT NULL,
    description TEXT,
    is_active BOOLEAN DEFAULT 1,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 系统设置表
CREATE TABLE settings (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    key TEXT UNIQUE NOT NULL,
    value TEXT NOT NULL,
    description TEXT,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 页面流程

### 初始化流程
1. 检测到无管理员用户 → 跳转 /init
2. 填写管理员账号密码
3. 配置数据库连接（MySQL/PGSQL）
4. 初始化完成 → 跳转后台

### 管理后台流程
1. /admin/login → 登录
2. /admin/dashboard → 控制台概览
3. /admin/datasource → 数据源CRUD
4. /admin/settings → 系统设置

## 安全设计
- 密码bcrypt加密存储
- Flask session管理登录状态
- 数据源密码AES加密存储
- 后台页面需要session验证
- 前台页面保持现有token验证
