## 一、项目概述

### 1.1 项目功能

基于企业微信智能表格API的商品价格查询系统，通过企微自建应用搭建查询表单，实时查询智能表格数据并展示产品报价。

### 1.2 核心功能

- 通过产品名、功率、柴油发动机型号、发电机型号查询商品价格
- 实时从企业微信智能表格读取数据
- 前端筛选并展示查询结果
- 自动计算报价（成本 + 成本×上浮指数%）
- 支持PC端和移动端自适应

### 1.3 项目结构

## 五、操作指南

### 5.1 环境准备

```bash
# 1. 复制配置模板
cp .env.example .env

# 2. 编辑.env文件，填入真实的配置信息
# corpid=YOUR_CORPID
# corpsecret=YOUR_CORPSECRET
# docid=YOUR_DOCID
# sheet_id=YOUR_SHEET_ID

# 3. 安装Python依赖
cd backend
pip install -r requirements.txt
```

### 5.2 开发环境启动

```bash
# 终端1：启动后端
cd backend
python app.py

# 终端2：启动前端（使用http-server）
cd frontend
npx http-server -p 8000 -c-1
```

### 5.3 生产环境部署

#### 后端部署（使用Gunicorn + Nginx）

```bash
# 1. 安装Gunicorn
pip install gunicorn

# 2. 创建启动脚本
cat > start.sh << 'EOF'
#!/bin/bash
export FLASK_ENV=production
gunicorn -w 4 -b 0.0.0.0:5000 --access-logfile - --error-logfile - backend.app:app
EOF
chmod +x start.sh

# 3. 配置Nginx
cat > /etc/nginx/sites-available/price-query << 'EOF'
server {
    listen 80;
    server_name your-domain.com;

    location /api/ {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location / {
        root /path/to/frontend;
        try_files $uri $uri/ /index.html;
    }
}
EOF
```

### 5.4 安全加固

```bash
# 1. 从Git历史移除敏感文件
git rm --cached .env
git commit -m "移除.env文件"
```
